MarvelListHeros
---------
MarvelListHeros is a simple app that shows you a list of the Marvel characters.

Developed with:
-------
- Kotlin;
- Clean Archicteture;
- Coroutines;
- ViewModel and LiveData;
- Retrofit;
- Dagger 2;
- Unit tests;
- UI tests;
- Detekt;
- Ktlint;

API
-------
This project use the Marvel Comics API.
https://developer.marvel.com/docs
